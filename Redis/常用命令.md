## 允许其他IP访问  只能在Test环境这么做, Pro禁止!

```
vim /usr/local/redis/etc/redis.conf
修改
protected-mode yes
为
protected-mode no


注释
#bind 127.0.0.1
#bind 127.0.0.1 ::1

重启redis
service redis-server restart

查看状态
ps -ef|grep redis
出现*:6379表示成功
/usr/local/redis/bin/redis-server *:6379

```
