**实时查看日志**

```
tail -f 日志名称
```

**以json方式查看日志**

> 需要安装jq软件
```
tail -f test.log | jq
```

**统计目录内文件总大小**
```
du -sh .
```

**统计目录内文件大小  按照文件从大到小排序**

**查看磁盘已用/可用空间**
```
df -h
```

**查看某个端口是否在使用**

**查看linux哪些端口允许被访问**
iptables -L -n --line-number
ACCEPT 表示允许

**查看某个程序是否在运行**
```
ps -ef | grep php-fpm
```

**统计某个程序的运行数量**
```
ps -ef | grep php-fpm | wc -l
```
