### beanstalk消息队列

```
#下载文件
wget https://github.com/beanstalkd/beanstalkd/archive/v1.10.zip
#解压
unzip v1.10.zip
cd beanstalkd-1.10
#编译并安装
make && make install
#创建目录
mkdir /data/runsoft/beanstalk
#启动 并设置后台运行
/usr/local/bin/beanstalkd -l 127.0.0.1 -p 11300 -b /data/runsoft/beanstalk &
```
